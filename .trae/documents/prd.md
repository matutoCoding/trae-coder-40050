## 1. 产品概述

精密铸造厂熔模铸造业务管理系统，用于铸造车间全流程数字化管理，覆盖蜡模制备、型壳制作、金属浇注到后处理的完整工艺链条。

- **核心目标**：实现熔模铸造生产全过程的数字化记录、质量追溯和工艺管控
- **目标用户**：铸造车间班组长、工艺工程师、质量检验员、生产管理人员
- **产品价值**：提升生产效率、保障产品质量、实现工艺数据可追溯

|------|----------|----------|

1. **蜡模压制模块**：蜡模压制记录、蜡件尺寸检验

3. **制壳挂砂模块**：涂料粘度监控、制壳层数管控、型壳干燥时长
4. **脱蜡焙烧模块**：脱蜡釜压力监控、型壳焙烧曲线
5. **合金熔炼模块**：中频炉熔炼配料管理
6. **浇注作业模块**：浇注温度记录
7. **清理打磨模块**：切割飞边记录、打磨抛光记录

## 3. 核心流程

### 3.1 生产主流程

熔模铸造生产遵循蜡模制备→模组组合→型壳制作→脱蜡焙烧→熔炼浇注→后处理的工艺顺序。

```mermaid
flowchart LR
    A["蜡模压制"] --> B["蜡件检验"]
    B --> C["模组焊接"]
    C --> D["制壳挂砂"]
    D --> E["型壳干燥"]
    E --> F["脱蜡焙烧"]
    F --> G["合金熔炼"]
    G --> H["浇注作业"]
    H --> I["切割飞边"]
    I --> J["打磨抛光"]
    J --> K["成品检验"]
```

### 3.2 数据录入流程

每个工序节点，操作人员选择对应工单，录入工艺参数和质量数据，系统自动记录时间戳和操作人员信息。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：工业蓝 (#1e40af) + 深灰 (#1f2937)，体现专业工业感
- **辅助色**：橙色 (#ea580c) 用于警告、绿色 (#059669) 用于合格、红色 (#dc2626) 用于异常
- **字体**：系统字体栈，中文使用微软雅黑/苹方，英文使用 Segoe UI
- **布局风格**：左侧导航 + 顶部状态栏 + 主内容区，卡片式布局
- **图标风格**：线性图标 (lucide-react)，简洁专业

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页仪表盘 | 数据概览 | 统计卡片、数据图表、工单列表、状态指示灯 |
| 工序记录页 | 各模块 | 表单输入、数据表格、状态标签、操作按钮组 |
| 详情弹窗 | 通用 | 分步展示、数据明细、时间线记录 |

### 4.3 响应式

- 桌面端优先设计，适配 1280px 及以上分辨率
- 平板端自适应布局，导航可折叠
- 表单和数据表格支持横向滚动

### 4.4 交互设计

- 数据录入表单分步引导，减少操作错误
- 关键操作二次确认，防止误操作
- 实时状态更新，异常醒目提示
- 表格支持筛选、排序、搜索
